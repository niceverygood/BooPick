## 🚀 시작하는 법

### 1단계: 폴더 통째로 부픽 프로젝트 루트에 복사

```bash
# Mac에서
cd ~/Projects/BooPick
cp -r ~/Downloads/boopick-pivot/* ./

# 또는
unzip ~/Downloads/boopick-v2-prompts.zip -d ./
```

이러면 부픽 프로젝트 루트에 `CLAUDE.md`, `prompts/`, `assets/` 가 들어옵니다.

### 2단계: Claude Code 실행

부픽 프로젝트 폴더에서:
```bash
cd ~/Projects/BooPick
claude
```

`CLAUDE.md` 가 프로젝트 루트에 있으면 Claude Code가 자동 인식합니다.

### 3단계: Phase 0 부터 순서대로 진행

각 Phase는 **독립된 채팅 세션**으로 진행 권장. 토큰 효율 + 컨텍스트 명확.

```
세션 1 (Phase 0): 기존 코드 리셋
  → /clear 후 새 세션
세션 2 (Phase 1): 매물 업로드
  → /clear 후 새 세션
세션 3 (Phase 2): AI 조건 파싱
  → ...
```

각 세션 시작 시 해당 Phase 파일을 통째로 복붙:
```
[클립보드에 PHASE_0_RESET.md 내용 복붙]
```

Claude Code가 자동으로 CLAUDE.md를 참조하면서 작업합니다.

## 💎 핵심 차별화 (피칭 시 강조 포인트)

다른 부동산 도구들이 못 하는 것:

| 일반 부동산 도구 | **BooPick** |
|---|---|
| 매물 정보 검색 | ✅ + **산업 관점 분석** |
| "이 매물 좋습니다" | ✅ "이 업종에 왜 좋은가" 자동 |
| 평/㎡ 단위 사장님이 직접 | ✅ 자동 변환 + 검증 |
| 사진 캡처 첨부 직접 | ✅ QR + 네이버부동산 자동 연결 |
| 의뢰 받을 때마다 1시간+ | ✅ **30초** |

## 🚨 주의 사항

### 절대 하지 말 것

1. **네이버부동산 자동 크롤링 금지** - 약관 위반
2. **매물 사진 자동 다운로드 금지** - 저작권 문제
3. **실 광고 텍스트 수정 후 재게시 금지** - 등록자 권한 침해

### 부픽이 합법적으로 하는 것

✅ 사용자가 직접 업로드한 엑셀 데이터 분석
✅ 매물번호 + 네이버부동산 URL 링크 (단순 식별자)
✅ QR 코드 (URL 인코딩, 자체 호스팅 X)
✅ AI 분석 텍스트 (원본 광고 인용 X)
