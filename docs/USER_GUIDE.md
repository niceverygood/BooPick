## 📍 부픽이 뭔가요?

사장님이 받은 매물 4만 건 엑셀 + 카톡으로 받은 의뢰 한 줄만 있으면,  
**30초 안에 AI가 분석해서 의뢰자에게 보낼 PDF 제안서로** 만들어 드립니다.

### 누구를 위해 만들었나
- 강남·판교 등 사옥 매물 다루시는 공인중개사 사장님
- 의뢰자한테 일일이 매물 찾아 비교표 만드는 시간이 아까운 분
- 결혼정보회사·법무법인 같은 특수 업종 의뢰가 자주 들어오는 분

### 핵심 가치
| 일반 워크플로우 | 부픽 |
|---|---|
| 엑셀에서 의뢰 조건 손으로 필터 | 카톡 메시지 그대로 붙여넣기 → AI 파싱 |
| 비교표 PPT/한컴으로 직접 그리기 | 9페이지 PDF 자동 생성 |
| 매물별 강·약점 직접 작성 | 산업 관점 4가지 포인트 자동 생성 (Pro) |
| 답사 1순위 직감으로 결정 | 0~100점 적합도 점수 자동 산출 |

## 📦 Step 1 — 매물 데이터셋 업로드

### 어떤 엑셀이 필요?

네이버부동산 광고 검색 결과 엑셀 다운로드 → 그대로 업로드.

권장 컬럼 (자동 매핑 됨):
```
매물번호 / 지역 / 공급m² / 전용m² / 해당층 / 전체층
보증금 / 월세 / 관리비 / 현재업종 / 추천업종
간략설명 / 설명 / 주소 / 사용승인일 / 중개사무소명
```

컬럼명이 정확히 일치하지 않아도 AI가 자동 매핑합니다.  
xlsx / xls / csv 모두 지원, **사이즈 제한 없음**.

### 업로드 흐름

1. **상단 메뉴 [업로드]** 클릭 → `/dashboard/upload`
2. 점선 박스 안에 **파일 드래그 앤 드롭** 또는 클릭해서 선택
3. 파일명·용량 확인 → **[다음 단계]** 클릭
4. 브라우저가 파일을 파싱 (수만 건도 5초 이내)
5. **컬럼 매핑 검토 화면**
   - 헤더 N개 / 매물 N건 / 매핑된 컬럼 N개 표시
   - 잘못 매핑된 컬럼은 드롭다운에서 수동 수정
   - 데이터셋 이름 변경 가능
6. **[💾 저장하고 검색하기]** 클릭
7. 진행률 바 (오렌지 → 그린)로 저장 상황 표시
   - 2,000건씩 배치 전송 → 4만 건 기준 약 20초
8. 완료 시 자동으로 검색 페이지로 이동

### 💡 팁
- 한 번 업로드한 데이터셋은 계속 재사용 가능. 같은 풀에서 의뢰만 다르게 분석.
- 새 매물이 들어오면 **새 데이터셋으로 업로드** (기존 데이터셋은 그대로 두기).
- 대시보드 **[내 데이터셋]** 카드에서 데이터셋 옆 빨간 [삭제] 버튼으로 제거 가능.

## 💬 Step 2 — 의뢰 메시지 입력

상단 [검색] 메뉴 또는 데이터셋 옆 [검색] 버튼 → `/dashboard/search`

### 4단계 멀티스텝 폼

```
[1 데이터셋] → [2 의뢰입력] → [3 조건검토] → [4 결과]
```

#### 1단계 — 데이터셋 선택
업로드한 데이터셋 중 하나 선택 → [다음 →]

#### 2단계 — 의뢰 메시지 입력
**카톡으로 받은 의뢰 메시지를 그대로 붙여넣으세요.**

예시:
```
결혼정보회사 / 삼성역 선릉역 선정릉역 삼성동 (청담X) /
사무실 140-200평 연층도 / 월관 최대 3천 / 직원 50명 /
구축X 20년 이내 / 입주 6-7월
```

또는 자연스럽게:
```
강남에 카페 25평쯤 보증금 5천 월세 300 즉시입주
```

→ **[🤖 AI로 분석하기]** 클릭

AI가 처리하는 것 (Claude Sonnet 4.5 사용):
- 비표준어 정규화 ("결정사" → "결혼정보회사")
- 제외 지역 추출 ("청담X" → 자동 exclude_regions)
- ㎡ ↔ 평 자동 변환
- 입주 시점 YYYY-MM 형식 정규화
- 보증금/월세/관리비 합산 조건 인식
- 주차 필수 여부, 연식 제한, 직원 수, 부가 요청

## ✏️ Step 3 — 조건 검토 + 수정

AI가 풀어준 15개 필드를 한 화면에서 확인·수정:

| 필드 | 예시 값 |
|---|---|
| 업종 | 결혼정보회사 |
| 지역 | 삼성역 / 선릉역 / 선정릉역 / 삼성동 (태그 추가·삭제) |
| 제외 지역 | 청담동 (빨간 태그) |
| 면적 | 140 ~ 200평, 연층 허용 ☑ |
| 임대료 | 월관 합계 3,000만원/월 |
| 보증금 | (비워두면 무제한) |
| 직원 수 | 50명 |
| 최대 연식 | 20년 이내 |
| 입주 시기 | 2026-06 |
| 주차 | 필수 ☑ |
| 추가 요청 | 상담실 15-20개 (태그) |

빠진 조건은 직접 추가, 잘못 파싱된 부분은 수정.

### 업종 선택 (Pro 차별화 포인트)

```
선택 안 함  |  🏢 일반 사무실  |  💍 결혼정보회사 PRO  |  ☕ 카페 (준비중)
```

- **일반 사무실** — 베이직·Pro 모두 가능. 표준 가중치만 적용.
- **결혼정보회사** — Pro 전용. 매물별 운영 관점 분석 4가지 포인트 자동 생성.
- **카페·학원·필라테스 등 6종** — V2 추가 예정 (준비중).

베이직 사용자가 결혼정보회사 클릭하면 토스트:  
> ⓵ 산업 관점 분석은 Pro 티어부터 이용 가능합니다.

→ **[🔍 매물 검색하기]** 클릭

## 📊 Step 4 — 매물 결과 + 점수 해석

```
검색 결과: 5건
40,795건 중 조건 부합 187건 → 상위 5건
```

### 비교표 (한 눈에)
```
#   면적   층   월관 합   주차    준공    점수
#1  191평  2층  2,800만   자주5   25.07   95
#2  151평  2층  2,500만   자주9   21.07   95
#3  151평  1~5  2,500만   자주11  21.07   95
#4  173평  2층  2,500만   자주4   23.10   80
#5  151평  5층  2,000만   자주4   23.11   80
```

### 매물별 점수 breakdown
각 매물 카드 클릭 또는 [▾ 점수 근거 보기]:

```
01 강남구 선릉역 / 1~5층 / 151평
   [90% 적합] [21.07 신축] [자주식 11대 ★] [통사옥]
   
   ├─ 면적   (30/30) ⭐ 151평 (의뢰 140-200평 정확)
   ├─ 임대료 (15/20) ✅ 합 2,500만 (예산의 83%)
   ├─ 연식   (20/20) ⭐ 21.07 신축 5년차
   ├─ 주차   (15/15) ⭐ 자주식 11대 (충분)
   ├─ 업종   (10/10) ⭐ "상담, 사무실, 오피스" 매칭
   └─ 기타   (5/5)   ⭐ 통사옥
   
                                              총 95/100
```

### 6차원 점수 만점 기준 (100점)
| 차원 | 만점 | 평가 |
|---|---|---|
| 면적 | 30 | 의뢰 범위 정확 부합 → 30, 경계 ±10% → 25, 연층 ±30% → 15 |
| 임대료 | 20 | 예산 70% 이하 → 20, 100%↓ → 15, +10% → 8 |
| 연식 | 20 | 신축 5년↓ → 20, 10년↓ → 15, max_age↓ → 10 |
| 주차 | 15 | 자주식 8대↑ → 15, 5대↑ → 10, 3대↑ → 5 |
| 업종 | 10 | 키워드 매칭 3개↑ → 10, 1개↑ → 5 |
| 기타 | 5 | "통사옥/통임대/렌트프리/단독사옥" 키워드 → +5 |

### Pro 전용 기능
- **매물 카드 클릭** → 네이버부동산 페이지 새 탭으로 열림 (실내 사진·지도·등록 부동산 연락처 확인)
- **각 매물별 산업 관점 분석** 4가지 포인트가 PDF에 자동 포함
- **QR 코드** 모바일 스캔으로 즉시 매물 확인

베이직은 매물번호 텍스트만 표시.

## 📄 Step 5 — PDF 리포트 다운로드

결과 화면 하단 **[📄 PDF 리포트 다운로드]** 클릭.

### 베이직 PDF (8페이지)
1. 표지 — 의뢰 조건 메타박스
2. 검토 요약 + 비교표
3-7. 매물 1~5 상세 (info-grid + 적합도 체크리스트)
8. 다음 단계 — 답사 일정 + 1순위 추천

### Pro PDF (9페이지)
1. 표지
2. **결혼정보회사 운영 관점 — 매물 평가 기준** (5가지 기준 카드)
3. 검토 요약 + 비교표
4-8. 매물 1~5 상세 + **산업 관점 분석 박스** (4가지 포인트)
9. 다음 단계

생성 소요 시간:
- 베이직: 30초 이내
- Pro: 60초 이내 (Claude 5건 병렬 분석 포함)

생성 완료 후 자동으로 다운로드, 파일명:  
`결혼정보회사_제안서_2026-05-11.pdf`

PDF 링크는 7일간 유효 — 의뢰자에게 카톡으로 그대로 전달 가능.

## ❓ 자주 묻는 질문

### Q. 의뢰자한테 PDF 안에 우리 사무소 이름·전화번호 들어가나요?
A. **사무소 이름만** 표지·매물 페이지 하단 "등록 부동산"에 표시됩니다.  
전화번호는 광고법 위반 우려로 직접 들어가지 않습니다.  
의뢰자가 매물 상세 정보가 필요하면 PDF 안의 QR 코드(Pro) 또는 매물번호로 네이버부동산 페이지에서 확인합니다.

### Q. 같은 데이터셋으로 의뢰자별 PDF 여러 개 만들어도 되나요?
A. 네, 데이터셋은 한 번 업로드하면 재사용. 의뢰가 바뀔 때마다 검색만 새로 돌리시면 됩니다.  
다만 **월 한도(베이직 5건 / Pro 50건)는 PDF 생성 횟수 기준**입니다.

### Q. 매물 풀이 매일 바뀌는데 어떻게 관리하나요?
A. 주 1~2회 네이버부동산에서 신규 매물 엑셀 받아서 새 데이터셋으로 업로드 권장.  
오래된 데이터셋은 대시보드 [내 데이터셋]에서 삭제 가능 (포함된 매물도 함께 삭제됨).

### Q. 청담동 같은 특정 지역 제외하고 싶어요.
A. 의뢰 메시지에 "**청담X**" 또는 "**청담제외**"로 쓰시면 AI가 자동으로 인식해서 결과에서 빼줍니다.  
검토 화면에서 "제외 지역" 태그를 직접 추가/제거도 가능합니다.

### Q. AI가 의뢰를 잘못 풀었어요. 어떻게 고치나요?
A. **Step 3 조건 검토 화면**에서 모든 필드 직접 수정 가능. 잘못 잡힌 지역은 X로 빼고, 누락된 조건은 추가하시면 됩니다.

### Q. PDF가 카톡 첨부 안 돼요.
A. 부픽 PDF는 모두 5MB 이내로 생성됩니다. 다운로드 후 카톡 [파일] 첨부로 전달 가능.  
만약 안 되면 PDF 링크(7일 유효)를 그대로 카톡 메시지로 보내시면 의뢰자가 클릭해서 열 수 있습니다.

### Q. 비밀번호를 잊었어요.
A. **카카오로 가입하셨으면** 비밀번호 없음 — 다시 [카카오로 로그인] 클릭.

## 🛠 문제 해결

### "Monthly limit reached" (429 에러)
이번 달 한도 도달. Pro 베타 신청하시거나 다음 달 1일까지 대기.

### "Pro tier required" (403 에러)
결혼정보회사 선택 후 검색했는데 베이직 티어. 일반 사무실로 바꾸시거나 Pro 베타 신청.

### "조건에 맞는 매물이 없습니다"
- 면적·임대료 범위가 좁을 수 있음. Step 3에서 조건 완화
- 지역 태그가 너무 제한적일 수 있음
- 데이터셋 자체에 해당 조건 매물이 없을 수 있음 (다른 데이터셋 시도)

### 업로드 진행률 바가 중간에 멈춤
브라우저 새로고침 → 대시보드에서 진행 중이던 데이터셋 삭제 → 다시 업로드.  
파일 사이즈가 너무 크면 분할해서 업로드 권장 (10만 건↑).

### PDF 생성 60초 넘게 안 끝남
- Pro 분석은 Claude API 5건 병렬이라 보통 30~50초. 90초 넘으면 새로고침 후 [📄 PDF 다운로드] 다시 클릭.
- 이미 생성된 PDF는 캐시되어 재생성 안 합니다 (한도 차감도 안 됨).

### 카카오 로그인 "KOE205" 에러
카카오 Developers 콘솔 측 설정 문제. 한대표(dev@bottlecorp.kr)에 문의.

## 📐 데이터 보안 & 개인정보

- **데이터셋 격리**: 사용자별 매물·리포트 완전 분리. 다른 사장님 매물은 절대 보이지 않음 (Supabase RLS 정책).

- **광고법 준수**: PDF에 임대인 전화·이름 등 PII 포함 안 함.
