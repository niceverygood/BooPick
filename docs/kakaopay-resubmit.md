# 카카오페이 재제출 가이드

> 2026-05 카카오페이 1차 검수 결과 "부동산 매물 비교 분석"으로 분류되어 민원다발업종 사유로 입점 거절.
> 본 문서는 재제출용 자료 패키지와 스크린샷 시퀀스를 정리한다.

## 1차 거절 사유 요약

1. **상품 식별 실패** — 사이트에서 결제 가능한 상품 구조가 명확하지 않음
2. **업종 분류 오인** — "부동산 매물 비교 분석"으로 보고 B2C 컨설팅 → 민원다발업종
3. **결제 경로 미구현** — "결제하기" 버튼까지의 전체 흐름이 없음

## 재제출 핵심 메시지

부픽은 **공인중개사용 B2B SaaS 도구**다. 매물 추천 · 중개 · 자문 서비스가 아니며,
사용자가 보유한 매물 데이터셋을 분석하는 소프트웨어 사용권을 월 구독으로 판매한다.

| 구분 | 1차 (오해) | 재제출 |
|------|-----------|--------|
| 상품 정의 | "매물 비교 분석" 컨설팅 | SaaS 도구 월 구독(월 49,000원) |
| 사용자 | 일반 소비자 | 공인중개사 사무소 (B2B) |
| 결과물 | "추천" | 본인 데이터 기반 분석 PDF |
| 책임 소재 | 회사가 추천 | 사용자 책임 명시 (약관·면책) |

## 카카오페이 요구 자료 — ③④⑤

### ③ 테스트 계정 (비회원 결제 불가 시)

```
URL:    https://boo-pick.vercel.app
```

### ④ 결제 흐름 스크린샷 (8장)

아래 순서로 스크린샷 캡처 후 PDF 1장으로 묶어 첨부:

| # | URL | 설명 |
|---|-----|------|
| 1 | `/` | 랜딩 — "공인중개사 매물 분석 SaaS" 포지셔닝 + 면책 박스 |
| 2 | `/pricing` | 가격 페이지 — Pro 49,000원/월, 구독 안내 + 환불 정책 링크 |
| 3 | `/login` | 로그인 화면 |
| 4 | `/checkout` (상단) | 결제 페이지 진입 — 상품/구매자 정보 입력 |
| 5 | `/checkout` (중간) | 결제 수단 선택 (카카오페이 · 카드 · 가상계좌) |
| 6 | `/checkout` (하단) | 약관 동의 3종 + 분석 도구 안내 박스 |
| 7 | `/checkout` (우측) | 주문 요약 카드 + "결제하기" 버튼 활성화 상태 |
| 8 | `/checkout/complete` | 결제 요청 접수 완료 페이지 |

#### 캡처 시 강조 포인트

- **#1**: 노란 "분석 도구 안내" 박스가 명확히 보이게 — "매물 추천 · 중개 · 알선 아님" 문구
- **#2**: 가격 "49,000원/월 (VAT 포함)" + "구독 안내" 박스
- **#7**: 우측 주문 요약에 **"부픽 Pro 구독 / 월 정기 / 49,000원"** 명확히 보이게
- **#8**: 완료 페이지 — 영업일 24시간 안내 + 환불 정책 명시

### ⑤ 이용약관 · 환불 정책 · 개인정보처리방침

| 페이지 | URL | PDF 출력 권장 |
|--------|-----|---------------|
| 이용약관 | `/terms` | ✓ |
| 환불 정책 | `/refund` | ✓ |
| 개인정보처리방침 | `/privacy` | ✓ |

→ 3개 페이지를 브라우저에서 PDF 출력 → 한 파일로 묶어 첨부.
