# 부픽 (BooPick) 사용가이드

> 공인중개사를 위한 AI 매물 매칭 도우미

---

## 부픽이 뭐예요?

**모바일 웹앱에서 손님 조건 한 줄 입력하면, 30초 안에 딱 맞는 매물 5개를 카드로 받아 손님께 카톡으로 즉시 공유하는 서비스입니다.**

이런 식이에요.

```
[손님 → 사장님 (카톡)]
"신사동에 25평 카페자리 보증금 1억 이하로 좀 알아봐주세요"

[사장님 → 부픽 웹앱 검색창]
신사동 25평 카페자리 보증금 1억 이하

[부픽 (30초 후, 웹앱 화면)]
🏠 매물 5건 매칭됐습니다

1. 가로수길 코너 1층 25평 / 보증금 8천 / 월세 350
   #코너 #테라스 #카페가능 #1층노출
   👉 매물 카드 보기 / 카톡으로 공유

2. 도산공원 1층 23평 / 보증금 9천 / 월세 380 [공동중개]
   #권리금없음 #즉시입주 #카페가능
   👉 매물 카드 보기 (○○부동산 매물)

3. ...

[사장님 → 손님 (카톡으로 매물 카드 링크 전달)]
부픽이 만든 깔끔한 카드 URL → 손님이 클릭하면 사진·정보·지도까지 다 보임
```

일일이 매물 장부 뒤지지 않아도 됩니다. **그 자리에서 손님에게 보낼 카드가 나옵니다.**

---

## 왜 만들었어요?

공인중개사 사장님들 일과를 보면:

- ☎️ 손님 전화 받음 → "강남에 1층 카페자리 있나요?"
- 📒 엑셀·노트·머릿속에서 매물 조회
- 🔍 조건 맞는 매물 골라서 손님에게 사진+설명 정리해서 카톡 발송
- 🔁 다른 사장님 매물도 알아봐야 함 → 단톡방에 "혹시 신사동 1층 카페자리…" 물어봄

**한 손님 응대에 평균 30~45분.** 매물 많은 강남권 사장님일수록 더 오래 걸립니다.

부픽은 이 과정을 **30초**로 줄입니다. 그리고 **다른 중개사 매물까지 한 번에** 찾아드립니다 (옵트인 기반).

## 핵심 기능 9가지

### 1. 🔎 자연어 매물 검색
"역삼동 사무실 30평 즉시입주 가능한 곳" 같은 한 줄을 그대로 던지면, AI가 알아듣고 매물을 찾아드립니다. 옵션 클릭, 필터 설정 안 해도 됩니다.

### 2. 🤖 매물 자동 태깅
새 매물을 등록하면 AI가 설명을 읽고 자동으로 태그를 답니다.
- **업종**: 카페·미용실·식당·사무실·학원·필라테스 등 22종
- **시설**: 테라스·복층·코너·엘리베이터·주차·통유리 등
- **입지**: 역세권·메인도로·코너자리·1층노출 등
- **상태**: 신축·리모델링·즉시입주·권리금없음 등

이렇게 태그가 달려 있으면 "테라스 있는 카페자리"로 검색해도 정확히 걸립니다. 일일이 수기로 분류 안 해도 됩니다.

### 3. 📱 모바일 웹앱 어디서든
별도 앱 안 깔아도 됩니다. **모바일 브라우저에서 `boopick.kr` 즐겨찾기 → 끝.** (홈 화면에 추가하면 앱처럼 아이콘으로도 실행)
- 사무실 PC 앞 안 가도 OK — 카페·차 안에서도 검색
- 검색 결과 카드를 카톡으로 손님께 1초 공유 (공유 링크 누르면 카톡이 자동으로 열림)
- 카톡 챗봇 연동은 베타 안정화 후 추가 예정 — 메시지로도 검색하고 싶으신 분들 기다려주세요

### 4. 🔔 자동매칭 알림
손님 조건을 부픽에 등록해두면, **그 조건에 맞는 새 매물이 풀에 들어올 때 알려드립니다.**
- "강남구 25평 이상 카페자리 보증금 1억 이하" 등록
- 다른 중개사가 그런 매물 새로 올림
- 부픽이 즉시 알림: 웹앱 인앱 알림 + 이메일 ("방금 들어온 신사동 매물이 손님 조건과 맞습니다")
- 알림 받고 사장님이 손님께 카톡으로 매물 카드 전달

손님 입장에서 부지런한 사장님이 됩니다. (Phase 2에서 알림톡으로도 받을 수 있게 추가 예정)

### 5. 🤝 공동중개 풀 — 부픽의 핵심
**다른 중개사 매물도 함께 검색됩니다 (서로 동의한 매물에 한해).**

기존: 단톡방에 "혹시 그 매물 있나요?" → 답 기다림 → 시간 낭비
부픽: 검색 결과에 다른 중개사 매물이 함께 나오고, 즉시 양타 협업 가능

- 매물 등록 시 **공유 풀에 올릴지 본인이 선택** (옵트인)
- 임대인 연락처 등 민감 정보는 **다른 중개사에게 절대 노출 X** (등록자만 볼 수 있음)
- 공유 매물에는 자동으로 [공동중개] 태그 + 양타 수수료 추정 표시

### 6. ✍️ 광고문구 AI 생성 (F7)
매물 정보 입력 → 채널별 광고문구 자동 생성.
- 네이버부동산·다방·직방·카카오·인스타용 각각 톤 다르게
- formal / casual / impact 톤 선택
- 매번 카피라이팅 고민 안 해도 됨

### 7. 🎙️ 음성메모 정리 (F8)
매물 둘러본 후 차에서 한마디 녹음:
> "방금 본 신사동 1층 매물 25평인데 코너자리고 테라스 5평 별도로 있고 사장님이 권리금은 받기 싫다고 하시고 이번 달 안에 빼고 싶다고…"

부픽이 받아쓰기(STT) → 정리 → 매물 등록 폼 자동 채움 + 태그까지 답아줌.

운전하면서 답변 못 하던 매물 정보가 즉시 시스템에 들어갑니다.

### 8. 📤 매물 공유 카드 (F9)
손님에게 매물 보낼 때 깔끔한 카드 한 장으로:
- 부픽이 매물별로 짧은 URL + 카톡 미리보기 이미지(OG) 자동 생성
- 사장님이 본인 카톡으로 손님께 링크 전달 → 손님이 클릭하면 사진·정보·지도 보임 (가입 불필요)
- 클릭 추적 → 손님이 어떤 매물에 관심 보이는지 확인

### 9. 📊 활동 통계
- 이번 달 매물 등록 수, 검색 수, 매칭 발송 수
- 공동중개 매칭 빈도 (내 매물 vs 다른 중개사 매물)
- 어떤 손님 조건이 매물 풀과 안 맞는지

매출 어디서 새는지 보입니다.

## 시작하기 (베타)

### 1단계. 가입
1. 모바일 브라우저에서 **`boopick.kr` 접속**
2. **"카카오로 시작하기"** 버튼 → 카카오 계정으로 로그인 (1초)
3. 사무소명·연락처 간단 입력 → 14일 체험 자동 시작
4. 홈 화면에 추가 (iPhone: 공유 → 홈 화면 추가, Android: 메뉴 → 앱 설치) — 이러면 앱 아이콘처럼 한 번에 실행

### 2단계. 첫 매물 등록 (3가지 방법 중 선택)
- **(가)** 모바일 웹앱에서 매물 사진 업로드 + "신사동 1층 25평 코너 테라스" 같은 설명 입력
- **(나)** 엑셀·CSV 일괄 업로드 (대량 매물 사장님용 — 데스크톱 권장)
- **(다)** 모바일에서 음성으로 매물 정보 녹음 (운전 중 활용)

부픽이 자동으로 정리해서 등록합니다.

### 3단계. 첫 검색
검색창에 손님 조건 그대로 입력:
> "강남 사무실 30평 즉시입주"

5건 매물 카드가 30초 안에 옵니다. 마음에 드는 카드 → **"카톡으로 공유"** 버튼 누르면 손님 카톡에 매물 카드 링크가 그대로 전달됩니다.

### 4단계. (선택) 공동중개 풀 참여
설정 → 공동중개 풀 옵트인 → 본인 매물 중 공유할 것 골라서 활성화. **공유한 만큼 다른 중개사 매물도 검색 결과에 같이 나옵니다.**

## 자주 묻는 질문

### Q. 다른 중개사한테 내 매물 정보 다 노출되는 거 아닌가요?
A. 아니요. **임대인 연락처·이름·메모는 등록자만 볼 수 있게 자동 마스킹됩니다.** 다른 중개사는 매물 위치·평수·금액·사진·태그만 보고, 관심 있으면 부픽 통해 등록자에게 공동중개 문의를 넣습니다.

### Q. 매물을 공유 풀에 안 올리면 부픽 의미 없는 거 아닌가요?
A. 본인 매물 검색만 해도 30초 매칭의 가치는 그대로입니다. 다만 **다른 사람 매물 검색은 공유한 사람만 가능** (give & take 구조)이라, 양타 수수료를 자주 받는 사장님은 공유가 이득입니다.

### Q. AI가 잘못 매칭하면요?
A. 매물마다 **매칭 근거 1줄**이 같이 표시됩니다 ("신사동 가로수길 1층 코너, 테라스 5평 별도. 미용실 운영 적합합니다."). 사장님이 보고 판단해서 손님께 보낼지 결정하면 됩니다. 잘못된 결과를 그대로 손님에게 자동 발송하지 않습니다.

### Q. 네이버부동산·다방 매물도 검색되나요?
A. **아니요.** 부픽은 부픽에 등록된 매물만 검색합니다. 외부 플랫폼 무단 크롤링은 법적 문제가 있어서 의도적으로 안 합니다. 대신 부픽 자체 풀(다른 중개사 옵트인 매물)이 풍부해질수록 가치가 커집니다.

### Q. 손님 정보는 어떻게 보호되나요?
A. 손님 이름·연락처는 **등록한 중개사 본인 사무소만** 볼 수 있습니다. 다른 사무소는 손님 존재 자체를 알 수 없습니다.

### Q. 카톡으로는 검색 못 하나요?
A. 베타 출시 시점(약 6주 후)엔 모바일 웹앱이 메인입니다. 검색·등록·관리 모두 웹앱에서 처리하시고, 매물 카드는 카톡으로 손님께 공유합니다. **카톡 챗봇으로 직접 검색하는 기능은 베타 안정화 후 Phase 2에서 추가 예정입니다.**

### Q. 사용 중 막히면?
A. 웹앱 우측 하단 **"문의" 버튼** 또는 이메일로 부픽 운영팀이 응대합니다 (Bottle Inc. 판교 사무실).
